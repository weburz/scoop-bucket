# Weburz Scoop Bucket

Scoop bucket for [Weburz](https://github.com/weburz) command-line tools.

## How do I install these tools?

Add the bucket once:

```powershell
scoop bucket add weburz https://github.com/weburz/scoop-bucket
```

Then install any tool from it:

```powershell
scoop install weburz/<tool>
```

For example:

```powershell
scoop install weburz/terox
```

## Documentation

`scoop help` or check [Scoop's documentation](https://scoop.sh).
